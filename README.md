# 🔮 拾光工具箱 LumiBox

收集散落在互联网的每一束实用之光

## 访问地址

- 国内备案域名：https://box.52eh.com
- Cloudflare Pages：https://lumibox.pages.dev

## 功能模块

- 📝 文本工具（JSON、Base64、正则等）
- 💻 开发工具（时间戳、颜色、Hash等）
- 🖼️ 图片工具（压缩、格式转换等）
- 🔍 查询工具（IP、域名、天气等）
- 🤖 AI 工具（对话、翻译等）

## 技术栈

- 前端：原生 HTML/CSS/JS + Tailwind CSS
- 后端：Cloudflare Workers (Hono)
- 部署：Cloudflare Pages + GitHub Actions
